# ADMA Absensi Kantor - Backend API

Sistem manajemen absensi karyawan berbasis Laravel dengan dukungan API mobile dan fitur geofencing.

## Tentang Aplikasi

ADMA Absensi Kantor adalah sistem manajemen absensi karyawan yang dibangun dengan Laravel framework. Aplikasi ini menyediakan fitur-fitur lengkap untuk mengelola absensi karyawan dengan teknologi modern, seperti:

- **Geofencing** - Validasi lokasi berbasis GPS untuk absensi yang akurat
- **Photo Upload** - Upload foto saat absen masuk dan pulang
- **Mobile API** - REST API lengkap untuk aplikasi mobile Flutter
- **Dashboard Admin** - Panel admin untuk mengelola karyawan dan laporan
- **Multiple Entry Methods** - Absen via mobile, manual input, atau bulk import
- **Reporting System** - Export laporan Excel dan PDF
- **Role-based Access** - Sistem peran admin dan karyawan

Laravel is accessible, powerful, and provides tools required for large, robust applications.
